# 学习进度 PWA

一个轻量级的渐进式 Web 应用，用于记录和管理多个学习目标的学习进度。数据保存在浏览器本地，支持离线使用与安装到桌面。

## 功能

- 创建多个学习目标（名称、描述、分类、截止日期）
- 两种进度模式：**百分比** 或 **学习时长（小时）**
- 记录每次学习（日期、增量、笔记）
- 总览统计：全部 / 进行中 / 已完成 / 累计学时
- 筛选：全部、进行中、已完成
- 点击目标卡片查看详情与学习记录
- PWA：可安装、可离线访问

## 快速开始

### 方式一：直接打开（开发调试）

双击 `index.html` 即可在浏览器中使用。  
注意：Service Worker 需要 HTTP 服务才能注册，直接打开文件时离线功能可能不可用。

### 方式二：本地服务器（推荐）

如果已安装 Python：

```powershell
cd "D:\桌面\工作预备役\学习进度"
python -m http.server 8080
```

然后在浏览器访问：<http://localhost:8080>

如果已安装 Node.js：

```powershell
npx serve .
```

### 安装为桌面应用

1. 通过本地服务器访问应用
2. 浏览器地址栏或菜单中出现「安装」提示，或点击页面右上角「安装应用」
3. 安装后可像原生应用一样从桌面/开始菜单启动

## 数据存储

所有数据保存在浏览器 `localStorage` 中，键名：`learning-progress-data`。  
清除浏览器站点数据会导致记录丢失，如需备份可导出 localStorage 内容。
